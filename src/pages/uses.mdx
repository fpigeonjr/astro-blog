## Code Editor and Terminal

I have used Visual Studio Code for many years as my code editor. These days I am experimenting with Zed and NeoVim. I am learning Vim this year and am loving the motions and key combos it unlocks.

## Hardware
- 2020 MacMini base 
- 2015 Macbook Air running Omarchy
- 2023 Macbook Pro (work provided)
- 27" 5K monitor

## Software
- Figma for UX design
- Slack for office communications
- Loqseq for note-taking
- Github for version control and project management
- Orion for web browsing
- Brave for web browsing

## Honorable Mentions

- [Bullet Journal](https://bulletjournal.com/) in a crappy notebook.
- [Cloudflare for Families](https://blog.cloudflare.com/introducing-1-1-1-1-for-families/) to filter content and improve network speeds.
- [Raspberry Pi][rpi] using [pi-hole][pihole] to filter ads on my network.
